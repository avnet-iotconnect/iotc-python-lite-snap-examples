
# /IOTCONNECT Python Lite SDK – Vision AI Demos (Raspberry Pi)

This repo mirrors the folder and onboarding style of Avnet's demos and adds a **Vision Smart‑Zone** example for Raspberry Pi 5 that streams JSON telemetry to the local /IOTCONNECT Unix sockets.

- Reference demos: https://github.com/avnet-iotconnect/iotc-python-lite-sdk-demos
- Python Lite SDK: https://github.com/avnet-iotconnect/iotc-python-lite-sdk

See **[`raspberry-pi/README.md`](raspberry-pi/README.md)** to get started.
